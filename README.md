## What's ifs?

A simple web app for instant file sharing.

## Features

- upload a file
- download a file
- remove a file
- list / purge all the stored files when logged as admin

## Installation

The recommended way is through the excellent [NPM](http://www.npmjs.org/):

    $ npm install

## Usage

    nodejs app
